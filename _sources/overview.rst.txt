################################################################################
Overview
################################################################################

Profkit is an open-source Python toolkit for using profilers with support for filtering, analysis and visualization.

The goal of Profkit is to provide a common interface for using the `cprofile <https://docs.python.org/3/library/profile.html>`_, `pyinstrument <https://pyinstrument.readthedocs.io/en/latest/>`_, and `yappi <https://github.com/sumerc/yappi/>`_ Python profilers, while also unifying and extending their functionality with respect to filtering, analysis, and visualization.
